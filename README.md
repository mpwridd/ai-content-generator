# AI Content Generator ✍️

Professional multi-platform content creation powered by **Mimo v2.5 Pro**.

## Features

- 🎯 **6 Platforms** — Twitter/X, Instagram, LinkedIn, TikTok, YouTube, Blog
- 🎨 **7 Tones** — Professional, Casual, Humorous, Educational, Persuasive, Storytelling, Inspirational
- 📏 **3 Lengths** — Short, Medium, Long
- ✨ **AI Enhancement** — One-click content improvement
- 📋 **Copy to Clipboard** — One-click copy
- 🏷️ **Keywords & Audience** — Advanced targeting options
- 📜 **Generation History** — Track recent outputs
- ⚡ **Quick Suggestions** — Pre-built topic ideas

## Tech Stack

- **Next.js 14** (App Router)
- **TypeScript** + **Tailwind CSS**
- **Lucide React** (icons)
- **Mimo v2.5 Pro** (AI model)

## Getting Started

```bash
npm install
npm run dev
```
